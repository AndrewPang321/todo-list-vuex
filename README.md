# Todo List
A simple todo list application to practise the use of Vue and Vuex.

## Frameworks
- Google Firebase
- Vue.js
- Vue x
- [Vuetify.js](https://vuetifyjs.com/en/)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
